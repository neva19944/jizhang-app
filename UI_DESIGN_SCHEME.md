# 记账应用 UI 设计方案 V4

## 设计概述

基于 `impeccable` 设计方法论，为记账应用打造了全新的**温暖极简主义**界面设计方案。

### 设计方向：温暖极简主义（Warm Minimalism）

**核心理念**：让记账变得愉悦、无压力，通过温暖的色调和简洁的布局，让用户愿意持续使用。

---

## 设计系统

### 🎨 色彩系统（OKLCH）

采用 OKLCH 色彩空间，确保感知均匀性：

- **主色调**：温暖蓝绿色 `oklch(60% 0.15 175)` — 传达信任感和增长感
- **收入色**：温暖绿 `oklch(65% 0.16 145)`
- **支出色**：温暖红 `oklch(62% 0.18 25)`
- **中性色**：带温暖色调的灰色（不用纯灰）— 避免冰冷感

**独特之处**：
- 所有中性色都带有微妙的温暖色调（hue: 80），创造 subconscious 的温暖感
- 避免使用传统的"金融蓝"和"警告红"，改用更温暖的色调
- 渐变背景增加深度和温暖感

### ✍️ 字体系统

- **主要字体**：Outfit — 具有个性的无衬线字体，现代且温暖
- **数字字体**：JetBrains Mono — 等宽字体，确保数字对齐
- **字体层级**：基于 1.333 (perfect fourth) 的模数比例尺

**独特之处**：
- 避免使用 Inter、Roboto 等随处可见的字体
- 数字使用等宽字体 + `tabular-nums` 特性，确保财务数据对齐
- 字体加载使用 `font-display: swap`，避免布局偏移

### 📐 空间系统

- **基础单位**：8px 网格
- **变化间距**：基于行高（1.5 × 16px = 24px）的垂直韵律
- **留白即内容**：充足的空间让数据呼吸

**独特之处**：
- 不使用统一的间距，而是根据内容节奏变化间距
- 使用 `clamp()` 实现流式间距（响应式）
- 容器查询 `@container` 支持真正的组件级响应式

### ✨ 动效设计

- **缓动函数**：指数缓动 `cubic-bezier(0.16, 1, 0.3, 1)`
- **有意义的动效**：只服务于功能，不炫技
- **微交互**：按钮按下的缩放、卡片悬停的上浮

**独特之处**：
- 避免使用弹性缓动（bounce/ease-out-back），显得不专业
- 动效时长遵循认知心理学：150ms（快）、300ms（正常）、500ms（慢）
- 使用 `transform` 和 `opacity` 实现流畅动画，避免布局抖动

## 页面设计

### 1️⃣ 首页（Home）

**设计亮点**：
- **英雄区域**：总资产数字使用超大字号（~50px），营造"一切尽在掌握"的感觉
- **渐变背景**：微妙的温暖渐变，增加深度感
- **装饰圆圈**：背景的半透明圆圈，创造视觉趣味
- **快速记账按钮**：固定在底部的 FAB，使用渐变背景 + 阴影，引人注目

**符合用户习惯**：
- 总资产、本月收入、本月支出一目了然
- 快速记账按钮触手可及
- 最近记录清晰展示

### 2️⃣ 记账页（Add）

**设计亮点**：
- **类型切换**：滑动背景动画，清晰指示当前状态
- **金额输入**：超大字号 + 居中对齐，数字优先
- **分类选择**：网格布局 + 激活状态的缩放动画
- **保存按钮**：渐变背景 + 悬停效果，主操作清晰

**符合用户习惯**：
- 打开页面即可输入金额（autoFocus）
- 快速金额按钮减少输入负担
- 分类图标 + 文字，清晰易懂

### 3️⃣ 统计页（Stats）

**设计亮点**：
- **周期切换**：滑动背景，与记账页保持一致
- **视图切换**：分类/趋势切换，清晰指示
- **饼图优化**：空心饼图 + 中心总计，数据层次清晰
- **图例列表**：背景卡片 + 百分比标签，易读性强

**符合用户习惯**：
- 默认显示本月数据
- 分类占比一目了然
- 趋势图帮助理解收支变化

### 4️⃣ 账户页（Accounts）

**设计亮点**：
- **添加表单**：滑入动画，不打断流程
- **账户卡片**：左侧彩色指示器 + 悬停效果
- **删除按钮**：悬停时显示，避免误操作
- **余额显示**：等宽数字 + 负余额红色背景

**符合用户习惯**：
- 账户列表清晰展示
- 添加账户流程简单
- 删除操作有确认提示

### 5️⃣ 更多页（More）

**设计亮点**：
- **区块标题**：左侧彩色边框，视觉层次清晰
- **表单卡片**：渐变背景 + 阴影，有质感
- **导出按钮**：卡片式设计 + 悬停效果
- **关于卡片**：顶部渐变装饰条，品牌感强

**符合用户习惯**：
- 预算设置简单直观
- 数据导出格式清晰
- 关于信息简洁明了

## 底部导航

**设计亮点**：
- **SVG 图标**：替代 emoji，更专业、更清晰
- **激活指示器**：底部小横线，微妙而清晰
- **毛玻璃效果**：`backdrop-filter: blur(10px)`，现代感强
- **安全区适配**：`env(safe-area-inset-bottom)`，适配 iPhone X+

**符合用户习惯**：
- 五个标签页覆盖核心功能
- 当前页面清晰指示
- 单手操作友好（拇指热区）

## 独特记忆点

为了让用户记住这个应用，我加入了以下独特设计：

1. **温暖的渐变背景**：每个页面都有微妙的温暖渐变，创造独特的视觉 identity
2. **数字等宽对齐**：财务数据使用等宽字体，专业且易读
3. **指数缓动动效**：所有动画都使用 `ease-out-expo`，流畅而不炫技
4. **左侧指示器**：账户卡片和区块标题的左侧彩色指示器，视觉统一
5. **装饰圆圈**：首页背景的半透明圆圈，增加趣味性

## 技术实现

### 设计 token 系统

```css
/* 色彩 token */
--primary: oklch(60% 0.15 175);
--income: oklch(65% 0.16 145);
--expense: oklch(62% 0.18 25);

/* 字体 token */
--font-primary: 'Outfit', system-ui, sans-serif;
--font-mono: 'JetBrains Mono', monospace;

/* 间距 token */
--space-1: 0.25rem;  /* 4px */
--space-2: 0.5rem;   /* 8px */
--space-3: 0.75rem;  /* 12px */
--space-4: 1rem;     /* 16px */
--space-5: 1.5rem;   /* 24px */
--space-6: 2rem;     /* 32px */

/* 动效 token */
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
--duration-fast: 150ms;
--duration-normal: 300ms;
```

### 响应式策略

- **移动端优先**：默认设计针对 375px 宽度
- **容器查询**：`@container (min-width: 400px)` 实现组件级响应式
- **流式字体**：标题使用 `clamp()` 实现流式缩放

### 性能优化

- **字体加载**：`font-display: swap` 避免 FOIT
- **动画性能**：只动画化 `transform` 和 `opacity`
- **CSS 变量**：减少重复，提高可维护性

---

## 可访问性

- **颜色对比度**：所有文本满足 WCAG AA 标准（4.5:1）
- **触摸目标**：最小 44px，符合移动端可访问性
- **键盘导航**：所有交互元素支持键盘访问
- **屏幕阅读器**：语义化 HTML + ARIA 标签
